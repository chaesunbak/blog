## AWS Certified Cloud Practitioner 자격증

| **시험시간** | 90분                                          |
| ------------ | --------------------------------------------- |
| **시험형식** | 65개 문항, 객관식 또는 복수응답               |
| **비용**     | 100 USD                                       |
| **시험방식** | Pearson VUE 테스트 센터 또는 온라인 감독 시험 |

### 시험 범위

- AWS 클라우드의 가치 설명
- AWS 공동 책임 모델 이해 및 설명
- 보안 모범 사례 이해
- AWS 클라우드 비용, 경제성 및 결제 관행 이해
- 컴퓨팅, 네트워크, 데이터베이스, 스토리지 서비스를 포함한 핵심 AWS 서비스  
  설명 및 포지셔닝
- 일반 사용 사례에 맞는 AWS 서비스 파악

### 시험 범위가 아닌것

- 코딩
- 클라우드 아키텍처 설계
- 문제 해결
- 구현
- 로드 및 성능 테스트

## 시험 팁

### 팁 1 : 언어 설정하기

대부분의 문제가 영어로 출제되며, 번역이 매끄럽지 않은 경우도 있다. 시험 언어를 한국어로 선택하더라도 영어 원문을 함께 볼 수 있다. **따라서 한국어로 신청하되, 번역이 어색하다고 느껴질 때는 영어 원문을 참고하는 것을 추천한다.** 한국어 신청 시 시험 편의 지원을 요청하면 30분의 추가 시간이 주어지지만, 시간이 부족한 시험은 아니기 때문에 필수는 아닌 것 같다.

### 팁 2 : 할인코드 확인하기

응시비용 100 USD는 결코 적은 금액이 아니다. 결제 당시(2025년 3월 31일) 환율로 무려 133,352원… 시기에 따라 응시료를 할인받을 수 있는 바우처를 뿌리므로, 결제 전 반드시 확인하는 것을 추천한다. **[Reddit](https://www.reddit.com/r/AWSCertifications/comments/1hlhj0k/2025_aws_vouchers_exam_discounts_coupons_other/)에서 관련 정보를 가장 빠르게 얻을 수 있는 것 같다.**

내가 응시할 당시에는 ETC 커뮤니티에 가입하여 활동 포인트를 얻은 후, 포인트를 할인 바우처로 교환하는 방법이 있었지만, 이 방법으로 바우처를 얻으려면 한 달 이상이 걸렸다... 따라서 정가를 지불했다.

(돈을 많이 버는 기업은 이유가 있는것 같다.)

### 팁 3 : 직관적으로 풀기

문제에 '인증서'라는 단어가 들어가면 'Certification'과 관련된 답을 고르는 것처럼, 직관적으로 푸는 것이 정답일 확률이 높은 것 같다.

## 시험 준비물

나는 테스트 센터를 방문하여 응시했다. 준비물은 다음과 같다.

신분증과 신용카드.

## 시험 과정

안내문에서는 시험 시작 15분 전에 도착할 것을 권고한다. 약 5분 정도의 확인 절차를 거친 후, 개인 물품을 사물함에 보관하고 시험실에 입장한다. 시험 중 화장실을 이용할 수 있으며, ILETS 등 다른 시험 응시자들과 함께 시험을 보게 됩다. 각자 시험 시간에 맞춰 자유롭게 퇴장할 수 있다. 나는 검토 시간을 포함하여 총 100분 중 약 50분을 사용했다. 시험 종료 후 바로 합격 여부를 확인할 수 있다.

## 후기

Q. 꼭 취득해야하는가?

A. 사실 난이도가 매우 낮아서 자격증으로서 의미는 별로 없는 것 같다. 클라우드 지식이 목표라면 강의를 듣는 것 만으로 충분할 것 같다. 하지만 나는 이후 상위 자격증을 도전할 의향이 있기 때문에 취득했다. AWS 자격증 하나를 취득하면 다른 AWS 자격증 시험 접수 시 50% 할인해주기 때문이다. 사실 결제를 안하면 움직이지 않는 고질병이 있는 것 같다. 선결제 후공부 방식에 너무 익숙해져있다. 결제와 뱃지에 움직이는 사람의 심리가 참…

Q. 클라우드 지식을 충분히 얻었는가?

A. 이번 자격증 취득으로 클라우드의 기본 개념과 장점, AWS에서 제공하는 서비스들에 대해서 알게되었다. 특히 기업의 입장에서 어떠한 AWS 서비스를 선택해야하는지를 주로 학습할 수 있었다.

하지만 여러 클라우드 서비스들이 있는 만큼 다른 클라우드 서비스 업체들의 서비스도 학습해보고, 프레미스 환경에 대한 지식도 쌓고 이를 클라우드 환경과 비교할 수 있어야, 클라우드 지식을 확실히 얻었다고 할 수 있을 것 같다.

이번 자격증을 취득하면서, AWS의 EC2를 사용해보기, 상위 AWS 자격증 취득하기, GCP 자격증 취득하기, 네트워크 관리사 자격증 취득하기, 라즈베리파이로 서버 구축해보기 같은 목표들을 설정할 수 있었다. 앞으로가 기대된다.
